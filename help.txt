قالب چند منظوره وبتی با بیش از 30 صفحه و دمو با طراحی یکپارچه و زیبا میتواند یکی از بهترین گزینه های ایجاد وبسایتتان باشد.

وبتی قالبی HTMl کاملا ریسپانسیو با رنگ های زیبا و گرم است داری طراحی و انمیشن های متنوع آماده برای استفاده شماست این قالب دارای صفحات لندینگ متنوع مانند لندینگ اپ، لندینگ کتاب، لندینگ برای استارت آپ ها و غیره است، این قالب همچنین دارای صفحات خاصی مانند فروشگاه، سئوالات متداول، گالری، بلاگ، هتل و غیره است. ین قالب با طراحی ریسانسیو و زیبا میتواند در طراحی وبسایتتان نقش به سزایی داشته باشد و در زمان شما برای ساخت وبسایت صرفه جویی کند.

در طراحی این قالب سعی شده است که تمامی نیاز های طراحی یک وبسایت بر طرف شود، قطعا سعی بر بهتر شدن کارکرد و طراحی قالب در آپدیت های بعدی وجود دارد که به صورت رایگان در اختیار خریداران وبسایت راستچین قرار میگیرد.

مزایای قالب وبتی:

دارای بیش از 30 صفحه آماده برای استفاده
استفاده از جدید ترین و قوی ترین کتابخانه css به نام Tailwind که توسط توییتر ایجاد شده است و قالب را چندین برابر سریعتر از متابخانه های دیگر مانند بوت استرپ کرده است
دارای قابلیت ریسانسیو و قابل اجرا روی تمامی دستگاه های هوشمند
دارای طراحی یک پارچه برای وبسایت شما
دارای صفحات مهم و ضروری برای هر وبسایتی 
دارای صفحات خاص
قابل استفاده برای وبسایت های لندینگ و وبسایت های عادی مانند بلاگ ها

نحوه استفاده از قالب برای وبسایتتان :

انتخاب فایل مورد نظر
انتقال فایل مورد نظر به همراه پوشه های css, dist,img و js به پروژه برای استفاده
تغییر آدرس دهی ها (در صورت استفاده از آدرس دهی شخصی در سایتتان)
تغییر محتوای قالب به محتوای دلخواهتان




نحوه تغییر در قالب و شخصی سازی آن:

برای ایجاد تغییر در قالب به شما پیشنهاد میکنم حتما با کتابخانه TailwindCss آشنایی داشته باشید
میتوانید موارد مورد نیاز برای ایجاد تغییرات در قالب را در پوشه develop پیدا کنید و به پروژه Tailwind خود اضافه کنید و از آن استفاده کنید
